Universal IST GitHub Project Rubric

Each major IST GitHub project is worth 20 points.

Category Points What I am looking for
Required Files & Evidence 5 Required files are included, named correctly, and easy to find. Screenshots, photos, audio, or code evidence are included when required.
Product / Technical Demonstration 5 The project works or clearly demonstrates the goal using the required tool, code, hardware, webpage, robot, music, or game feature.
README Explanation & Reflection 5 The README explains what was made, how it works, problems solved, and what could be improved.
Professional Habits & Process 5 The student followed directions, used class tools responsibly, committed work, used appropriate content, and showed effort/problem-solving.

Score Guide

  • 18–20: Strong / exceeds expectations
  • 15–17: Proficient / meets expectations
  • 12–14: Developing / partially meets expectations
  • 1–11: Incomplete or needs major revision
  • 0: Not submitted
