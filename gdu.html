Course Overview

Game Development in Unity is the third course in the Game Design pathway. Students will apply programming, design, animation, storytelling, teamwork, and testing skills to create interactive game projects.

Major Focus Areas

  • Game design process
  • Unity and C# programming
  • Animation and simulation
  • Level design and player experience
  • Playtesting, debugging, and iteration
  • Unity credential preparation
