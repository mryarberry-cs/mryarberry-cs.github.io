Course Overview

Introduction to Software Technology is the first course in the pathway. Students will learn how computer science can be used to create websites, music, hardware projects, robots, and games.

Major Projects

  1. GitHub Intro
  2. Portfolio Setup
  3. Web Creator Project
  4. EarSketch Music Project
  5. micro:bit Hardware Project
  6. Maqueen Robot Challenge
  7. Mini Game Project
  8. Final Portfolio Showcase

18-Week Pacing

Weeks Focus Project
1 GitHub setup and classroom expectations GitHub Intro
2 Portfolio setup, TSA intro, digital citizenship Portfolio Setup
3–5 HTML, CSS, web design, accessibility Web Creator Project
6–7 JavaScript and coded music EarSketch Music Project
8–9 micro:bit input, output, and sensors micro:bit Hardware Project
10–12 Maqueen robot movement, sensors, and debugging Maqueen Robot Challenge
13–16 Browser-based mini game design Mini Game Project
17–18 Portfolio organization, TSA connection, showcase Final Portfolio Showcase

Tools Students Will Use

  • GitHub Classroom
  • GitHub Codespaces
  • CodeHS
  • EarSketch
  • Microsoft MakeCode
  • micro:bit
  • Maqueen robots

Family Note

Students will use GitHub to document major projects and build a portfolio. Private assignment links and student submissions will not be posted on this public website.
