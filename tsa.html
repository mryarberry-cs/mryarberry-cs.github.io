What is TSA?

TSA is a career and technical student organization where students apply technology, leadership, teamwork, communication, and problem-solving skills through competitive events and chapter activities.

How TSA Connects to Class

In class, students will complete projects that build skills connected to TSA events. Not every class project is an official competition entry, but each project helps students practice documentation, creativity, teamwork, and technical problem solving.

Possible Event Connections

  • Career Prep: portfolio and career readiness artifacts
  • Webmaster: web design and accessibility projects
  • Music Production: coded music and audio design
  • Coding: programming fundamentals and problem solving
  • Software Development: planning, coding, testing, and documentation
  • Video Game Design: mini games and Unity game development
  • Robotics: micro:bit and Maqueen robot challenges

Important Competition Note

Official TSA competition entries must follow current TSA rules, event guidelines, copyright expectations, and academic integrity requirements. Competition work may have stricter rules than normal class practice.
