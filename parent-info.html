Course Purpose

Students in this pathway learn how computer science can be used to create websites, music, hardware projects, robots, games, and digital portfolios.

How Students Submit Work

Students will use approved classroom systems to receive assignments. Major project evidence may be documented in GitHub Classroom using code files, screenshots, photos, audio files, and written reflections.

Privacy and Safety

Students are expected to use school-appropriate usernames and avoid posting personal information online. Student grades, private assignment links, and individual student submissions will not be posted on this public website.

TSA

Students will be introduced to TSA opportunities connected to computer science, game design, software development, web design, music production, and robotics. Official competition participation may depend on student interest, eligibility, event limits, deadlines, and local requirements.
