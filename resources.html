Classroom Tools

  • CodeHS: coding practice, lessons, and skill checks
  • GitHub Classroom: project submissions and portfolios
  • GitHub Codespaces: browser-based coding environment
  • EarSketch: music creation with code
  • Microsoft MakeCode: micro:bit, Maqueen, and hardware coding
  • Unity: game development and simulation work in GDU

Project Evidence

Students will often submit project evidence using a README file, code files, screenshots, photos, audio files, or reflection responses.

Privacy Reminder

Students should not post personal information, private contact information, grades, or private assignment links on public websites.
